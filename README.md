# 📈 Crypto Watch Tower

> A real-time cryptocurrency dashboard powered by the CoinGecko API — built with React, Redux Toolkit & Recharts.

## ✨ Features

- 📊 **Interactive charts** — Price evolution visualized with Recharts
- 🔄 **Real-time data** — Live crypto prices fetched from the CoinGecko API
- 🗂 **Centralized state** — Global state management with Redux Toolkit
- 🔍 **Multi-coin tracking** — Monitor multiple cryptocurrencies at a glance
- 📱 **Responsive** — Works seamlessly on desktop & mobile

---

## 🛠 Tech Stack

| Layer | Tech |
|---|---|
| Frontend | React 17, React Router v6 |
| State Management | Redux Toolkit, React-Redux |
| Charts | Recharts 2 |
| HTTP | Axios |
| Styling | Sass |
| API | CoinGecko (free, no key required) |

---

## 🚀 Getting Started

### Prerequisites

- Node.js >= 14

### Installation

```bash
git clone https://github.com/Tmekz/Redux-Crypto-Watch-Tower.git
cd Redux-Crypto-Watch-Tower
npm install
```

### Run locally

```bash
npm start
```

> No API key needed — CoinGecko's public API is free and open. 🎉

## 📁 Project Structure

```
src/
├── components/       # Reusable UI components (CoinCard, Chart, Header...)
├── pages/            # Route-level pages (Dashboard, CoinDetail...)
├── store/            # Redux store, slices & async thunks
├── services/         # Axios config & CoinGecko API calls
└── styles/           # Global Sass styles
```
